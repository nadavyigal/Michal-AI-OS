# אוטומציית Wix Analytics

פרויקט Antigravity לדוחות אתר Wix של מיכל.

## מטרה

לא שמיכל תיכנס ידנית כל יום לדשבורד Wix, אלא שתהליך אוטומטי יביא נתונים ויפיק סיכום.

## שלבי הקמה

### אפשרות א — API רשמי של Wix

צריך:

- Wix API Token
- Site ID / Account ID לפי דרישות Wix
- הרשאת Site Analytics Read

לשים בקובץ `.env` לפי `.env.example`.

### אפשרות ב — דוחות CSV אוטומטיים

אם API מורכב מדי בהתחלה:

1. להגדיר ב-Wix שליחת דוחות Analytics למייל.
2. להוריד CSV לתיקיית `data`.
3. לתת לסוכן לקרוא את ה-CSV ולעדכן את הסיכום.

## משימה ראשונה לסוכן

```text
בדוק את הפרויקט. אם יש .env עם פרטי Wix, תכנן משיכת נתונים מה-API. אם אין, חפש CSV בתיקיית data וסכם אותו בעברית.
```
