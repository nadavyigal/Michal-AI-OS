# פרומפטים — אוטומציית Wix

## הקמה (פעם אחת)

```text
בדוק את הפרויקט. המטרה: דוח ביצועים יומי אמיתי מ-Wix, לא מומצא.
1. אם יש .env עם WIX_API_TOKEN ו-WIX_SITE_ID: השלם את scripts/fetch_wix_analytics_api.py
   לפי תיעוד Wix Analytics Data API (visits, unique visitors, page views, leads), הרץ,
   ושמור outputs/wix_analytics_raw.json ואז outputs/summary_latest.md בעברית.
2. אם אין API: חפש CSV ב-data, הרץ scripts/summarize_wix_csv.py, הפק summary_latest.md.
אל תמציא מספרים. אם אין נתונים, כתוב "אין נתונים — צריך API token או CSV".
```

## בוקר — דוח יומי

```text
הפק דוח ביצועים יומי בעברית מהנתונים האמיתיים בלבד (API אם קיים, אחרת CSV האחרון).
כלול: תאריך, כניסות, מבקרים ייחודיים, לידים/WhatsApp, שיעור המרה, עמוד מוביל,
מקור תנועה מוביל, שינוי מול אתמול, תובנה אחת, פעולה אחת.
שמור ל-outputs/summary_latest.md ותן סיכום של חמש שורות. אם אין נתונים אמיתיים — אמור זאת.
```

## דרישות לנתונים אמיתיים

- Wix API token עם הרשאת Site Analytics Read + WIX_SITE_ID ב-.env, או
- CSV יומי שיוצא מ-Wix Analytics למייל ומורד לתיקיית data.
