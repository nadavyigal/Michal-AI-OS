# Story 3 - דוח ביצועים יומי לאתר Wix

## מטרה

שמיכל תקבל בכל יום תמונה פשוטה וברורה:

- כמה אנשים נכנסו לאתר.
- כמה פניות / לידים התקבלו.
- כמה רכישות / מכירות היו.
- כמה כסף נכנס.
- מאיפה הגיעו הגולשים.
- איזה עמודים עבדו הכי טוב.
- האם יש שיפור או ירידה לעומת הימים הקודמים.

## הפתרון המומלץ

להתחיל בפתרון יציב ופשוט:

1. להגדיר ב-Wix מנוי יומי לדוחות Analytics רלוונטיים.
2. לקבל את הדוחות במייל כ-CSV.
3. לעדכן פעם ביום את האקסל / Google Sheet המרכז.
4. אחרי שבוע-שבועיים, אם זה עובד, אפשר להפוך חלק מהתהליך לאוטומטי יותר.

## דוחות Wix שצריך להפעיל

ב-Wix Dashboard:

1. Analytics
2. All Reports
3. לבחור דוח
4. Subscribe / Export

דוחות מומלצים:

- Traffic Overview - כניסות, סשנים, צפיות באתר.
- Sales Overview - מכירות, הזמנות, הכנסות. רלוונטי אם המכירה עוברת דרך Wix.
- Form Submissions over Time - פניות / טפסים.
- Form Submissions by Traffic Source - מאיפה הגיעו הלידים.
- Top Pages / Page Visits - אילו עמודים מושכים הכי הרבה תנועה.

## KPIים יומיים

### מדדים ראשיים

| מדד | הגדרה | למה זה חשוב |
|---|---|---|
| Visitors | מספר מבקרים באתר | כמה אנשים נחשפו למיכל |
| Sessions | מספר ביקורים באתר | כמה פעמים אנשים נכנסו |
| Page Views | מספר צפיות בעמודים | רמת עניין כללית |
| Leads | מספר פניות / טפסים | כמה אנשים השאירו פרטים |
| Orders | מספר רכישות | כמה מכירות בפועל |
| Revenue | הכנסות יומיות | כסף שנכנס |
| Lead Conversion Rate | Leads / Visitors | כמה מהמבקרים הפכו לליד |
| Sales Conversion Rate | Orders / Visitors | כמה מהמבקרים קנו |
| Average Order Value | Revenue / Orders | ממוצע הכנסה להזמנה |

### מדדים תפעוליים

| מדד | שימוש |
|---|---|
| Top Traffic Source | להבין מאיפה כדאי להביא עוד תנועה |
| Top Page | להבין איזה עמוד מייצר עניין |
| Notes / Actions | החלטה יומית קצרה: מה לשפר, לבדוק או לפרסם |

## שגרה יומית מומלצת

כל בוקר:

1. לפתוח את המייל היומי מ-Wix.
2. להוריד / לפתוח את קובץ ה-CSV.
3. להעתיק את המספרים המרכזיים לשורה חדשה ב-Daily Input.
4. לבדוק את ה-Dashboard.
5. לכתוב שורת פעולה אחת:
   - מה עבד?
   - מה ירד?
   - מה צריך לבדוק?

## שגרה שבועית מומלצת

פעם בשבוע:

- לבדוק איזה מקור תנועה הביא הכי הרבה לידים.
- לבדוק איזה עמוד הביא הכי הרבה עניין.
- לבדוק האם יחס ההמרה משתפר או יורד.
- להחליט על פעולה אחת לשבוע הבא: פוסט, קמפיין, שינוי כותרת, שינוי CTA, חידוד עמוד נחיתה.
