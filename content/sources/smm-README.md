# מנהל הרשתות החברתיות — מיכל

זו שכבת **האסטרטגיה** שמתבססת על העוזרות שבנינו (Track C). שם בנינו את העוזרות עצמן
(עוזרת תוכן/קול, עוזרת ניוזלטר, עוזרת ריפרפוז, עוזרת גבייה); כאן בנינו את **האסטרטגיה**
שהן מבצעות בפועל — מי קהל המטרה שלך, מה את מייצגת, על מה את כותבת, מתי, עם איזה פתיח,
איך מתקשרים עם הקהילה, ואיך מודדים הצלחה.

נבנה על בסיס כל מה שכבר מצאנו עלייך (מסמך הקול, התובנות האמיתיות מהאתרים שלך, חלוקת
התוכן האמיתית שלך) — לא תבניות גנריות.

## איך זה מתחבר לעוזרות

```
מנהל הרשתות (התיקייה הזו — מה לפרסם, מתי, ולמה)
        │
        ▼
עוזרת תוכן/קול (Content/Voice Gem — איך לכתוב את זה, בקול שלך באמת)
        │
        ▼
פוסט מוכן
```

הלוח השנתי (קובץ 04) ובנק הפתיחים (קובץ 05) נותנים את **הרעיון**. כשמעבירים את הרעיון
דרך עוזרת התוכן/קול, מקבלים את **הטקסט המוכן לפרסום**, בקול שלך.

## דברים חשובים לדעת

- **שלב 7 עדיין לא רץ באמת.** אין עדיין נתוני אינסטגרם/פייסבוק אמיתיים. זו תבנית
  מוכנה לרגע שיהיו נתונים, לא ניתוח סופי.
- **הזהות הוויזואלית (קובץ 02) זמנית**, בהמתנה לאישור שלך הערב — מצאנו שני סטים שונים
  של עיצוב באתרים הקיימים, וצריך לדעת מה הנכון/עדכני.
- **חלוקת הקהל (קובץ 01) מעלה שאלה אמיתית, לא רק תובנה**: לנהל גם תוכן זוגיות וגם תוכן
  ניהול באותו פיד יכול לדלל את שניהם. זו שאלה פתוחה אלייך, לא קביעה.
- **חוק הפרטיות תקף גם כאן.** הטקטיקה ב"שיתוף תובנות מלקוחות" (קובץ 06) חייבת להישאר
  במישור הדפוס הכללי — בלי שום פרט מזהה, אף פעם.
