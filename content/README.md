# מערכת תוכן של מיכל

פרויקט Antigravity לתוכן שיווקי בלבד.

## מה לשים כאן

בתיקיית `sources`:

- מסמך הקול של מיכל
- קבצי Social Media Manager
- עמוד נחיתה מחוברים מחדש
- עמוד עסקים משפחתיים
- תמלולי סרטונים ציבוריים

בתיקיית `outputs`:

- פוסטים מוכנים
- ניוזלטרים
- רעיונות לקמפיינים
- גרסאות משופרות לעמודי נחיתה

## משימה ראשונה לסוכן

```text
קרא את כל המקורות בתיקיית sources. בנה לי מפת תוכן בעברית:
1. 5 עמודי תוכן מרכזיים
2. 20 רעיונות לפוסטים
3. 10 הוקים
4. 5 רעיונות לניוזלטר
5. איך להשתמש בעמודי הנחיתה כחומר לתוכן שיווקי
```
