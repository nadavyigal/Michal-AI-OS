# משימות תוכן מוכנות

## להפוך עמוד נחיתה לפוסטים

```text
קח את עמוד הנחיתה שב-sources והפוך אותו ל:
1. 5 פוסטים לאינסטגרם
2. 5 סטוריז
3. 3 הוקים לסרטון
4. ניוזלטר קצר
הכול בקול של מיכל, בעברית, בלי סגנון שיווקי מוגזם.
```

## לבנות חודש תוכן

```text
על בסיס כל המקורות, בנה תוכנית תוכן ל-30 יום.
לכל יום תן: נושא, פורמט, הוק, גוף קצר, CTA וכיוון ויזואלי.
```

## שיפור פוסט

```text
שפר את הפוסט הבא כך שישמע יותר כמו מיכל: אישי, ישיר, חם, בוגר ולא גנרי.
```
