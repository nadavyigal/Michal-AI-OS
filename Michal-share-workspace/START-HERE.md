# Start Here — Michal AI OS Share Workspace

This folder collects the working materials for Michal's AI Operations Pilot.

## What we are trying to do

Build a privacy-safe Gemini operating system for Michal's coaching business. The first version is not automation that sends messages or touches client data. It is a controlled set of Gemini Gems that help draft billing summaries, content, newsletters, and repurposed posts while keeping client personally identifiable information out of Gemini.

The core idea:

- One shared "Michal's Voice" document controls tone, style, language, and visual direction.
- Four Gems use that foundation: Billing, Content/Voice, Newsletter, and Repurpose.
- Every Gem starts with the same privacy preamble.
- Client names, phone numbers, emails, and other PII stay out of Gemini. Michal uses aliases in AI, then reconnects real names offline.
