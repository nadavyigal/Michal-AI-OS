## פרויקט 01 — content-os

### הקמה (להריץ פעם אחת)

```text
קרא את כל הקבצים בתיקיית sources, כולל מסמך הקול, קבצי ה-SMM ושני עמודי הנחיתה.
בנה קובץ outputs/מפת-תוכן.md בעברית עם:
1. חמישה עמודי תוכן מרכזיים (Content Pillars) שמתאימים לקול של מיכל.
2. עשרים רעיונות לפוסטים, כל אחד עם הוק פותח.
3. חמישה רעיונות לניוזלטר.
4. איך להשתמש בשני עמודי הנחיתה כחומר לפוסטים.
אל תמציא עובדות, מספרים או תוצאות שלא מופיעים במקורות. אם משהו חסר, סמן [לאימות].
```

### פרומפט יומי — טיוטת תוכן

```text
נושא היום: <כתבי כאן נושא או רעיון>.
על בסיס מסמך הקול וקבצי ה-SMM ב-sources, כתוב טיוטת פוסט בעברית:
- הוק פותח
- גוף
- CTA אחד ברור
- כיוון ויזואלי
- איפה להשתמש: פוסט / סטורי / ניוזלטר
שמור ל-outputs/ עם תאריך בשם הקובץ. אל תמציא נתונים.
```

### לפני פרסום — בדיקת RED TEAM

מעבירים את הטיוטה למחברת `מיכל - RED TEAM` ב-NotebookLM ומבקשים בדיקה. רק אחרי "מוכן לפרסום" מפרסמים.

## פרויקט 03 — wix-analytics

### הקמה (להריץ פעם אחת) — לחבר מקור נתונים אמיתי

```text
בדוק את הפרויקט. המטרה: דוח ביצועים יומי אמיתי מ-Wix, לא מומצא.

שלב 1 — בדוק אם קיים קובץ .env עם WIX_API_TOKEN ו-WIX_SITE_ID.
אם יש:
- קרא את scripts/fetch_wix_analytics_api.py. זהו שלד בלבד.
- היכנס לתיעוד הרשמי של Wix Analytics Data API והשלם את הסקריפט לפי סוגי המדידה האמיתיים (visits, unique visitors, page views, leads/contacts), כולל טווח תאריכים של אתמול.
- הרץ אותו, שמור JSON גולמי ל-outputs/wix_analytics_raw.json, ואז הפק outputs/summary_latest.md בעברית.

שלב 2 — אם אין .env או שה-API נכשל:
- חפש קובץ CSV בתיקיית data (דוח Analytics שיוצא מ-Wix למייל).
- הרץ scripts/summarize_wix_csv.py והפק outputs/summary_latest.md בעברית.

אל תמציא מספרים. אם אין נתונים אמיתיים, כתוב במפורש "אין נתונים — צריך API token או CSV".
```

### פרומפט בוקר — דוח יומי

```text
הפק דוח ביצועים יומי לאתר של מיכל בעברית, מהנתונים האמיתיים בלבד (API אם קיים, אחרת CSV האחרון ב-data).
כלול:
- תאריך
- כניסות ומבקרים ייחודיים
- לידים / הרשמות / לחיצות WhatsApp
- שיעור המרה
- עמוד מוביל ומקור תנועה מוביל
- שינוי משמעותי לעומת אתמול
- תובנה אחת + פעולה אחת מומלצת
שמור ל-outputs/summary_latest.md ותן לי סיכום של חמש שורות. אם אין נתונים אמיתיים, אמור זאת במפורש ואל תמציא.
```

הערה: כדי שהדוח יהיה "נתונים אמיתיים" צריך אחד משניים — Wix API token עם הרשאת Site Analytics Read, או CSV יומי שיוצא מ-Wix. עד שיש אחד מהם, הדוח ירוץ במצב "אין נתונים".
