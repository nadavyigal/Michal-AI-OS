# הפעלת תצוגות עמודי הנחיתה במחשב של מיכל

יש שני קבצי HTML לתצוגה:

- `connected-short-preview.html` — דף נחיתה מקוצר למחוברים מחדש
- `family-business-preview.html` — עמוד שיווקי לעסקים משפחתיים

## אפשרות 1 — פתיחה פשוטה

1. להוריד את שני קבצי ה-HTML מ-Google Drive למחשב של מיכל.
2. ללחוץ פעמיים על כל קובץ.
3. הקובץ ייפתח בדפדפן.

זה מספיק כדי לראות את העיצוב והתוכן.

## אפשרות 2 — פתיחה דרך localhost

אם רוצים לראות את זה כמו אתר מקומי:

1. לשים את שני קבצי ה-HTML באותה תיקייה במחשב.
2. לפתוח Terminal באותה תיקייה.
3. להריץ:

```bash
python3 -m http.server 8123
```

4. לפתוח בדפדפן:

```text
http://localhost:8123/connected-short-preview.html
```

או:

```text
http://localhost:8123/family-business-preview.html
```

## הערה חשובה

אלה לא עמודי Wix חיים עדיין.
אלה תצוגות HTML כדי שמיכל תראה את הכיוון הוויזואלי והתוכן לפני שמטמיעים ב-Wix.

כדי להפוך אותם לעמודי Wix:

1. לפתוח Wix.
2. לשכפל עמוד קיים או ליצור עמוד חדש.
3. להעתיק את הטקסט והמבנה מהקבצים.
4. להוסיף תמונות/כפתורים ב-Wix.
5. לחבר CTA לרכישה, טופס או כרטיס ביקור.
